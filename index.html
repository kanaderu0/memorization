プログラミング言語集

～暗記機能付き～

  • HyperText Markup Language（HTML）
    Webページの構造を定義するためのマークアップ言語
  • Cascading Style Sheets
    （CSS）
    Webページのデザインやレイアウトを設定するための言語
  • JavaScript
    Webページに動的な機能を追加するためのプログラミング言語
  • C言語
    1972年にアメリカのAT&Tベル研究所で開発されたプログラミング言語
  • Java
    1995年にSun Microsystems（現在はOracle社）によって開発された汎用性の高いプログラミング言語およびコンピューティングプラットフォーム
  • C#
    マイクロソフトが開発したオブジェクト指向の汎用プログラミング言語で、.NETプラットフォーム上で動作する
